Course Overview

课程概况

掌握企业级数据库运维核心技能，从安装配置到性能优化的全流程实战

🎯

课程目标

掌握MySQL数据库的安装配置、权限管理、日志分析、备份恢复、性能监控、主从复制、集群运维等核心技能，培养企业级DBA思维与实战能力。

📖

参考教材

《MySQL数据库运维与管理》
邓文达、邓河 主编
ISBN: 978-7-115-60614-3

💻

学习环境

CentOS 7 + MySQL 8.0
Windows + Linux 双平台实践
云服务器 + Docker容器

👥

面向对象

高职院校计算机软件技术专业
大二学生（大数据应用开发方向）
对数据库运维感兴趣的学习者

⚡

核心技能

数据库安装与配置 · 权限与安全管理
日志分析与故障排查 · 数据备份与恢复
性能监控与优化 · 高可用集群部署

🌟

教学特色

行动导向：跟着做就能学会
场景驱动：天天电器业务案例贯穿始终
循序渐进：P1→P10 逐级深入

Learning Path

学习路径

从数据库认知到智能运维，10个项目带你系统掌握DBA核心技能

P1 数据库认知 4课时

了解数据库运维工作，认识DBA岗位职责，就业前景分析

P2 安装配置 4课时

Linux/Windows双平台安装MySQL，配置服务环境

P3 权限安全 4课时

用户管理、权限授予与收回、远程访问控制、安全加固

P4 日志分析 6课时

错误日志、慢查询日志、二进制日志分析与故障排查

P5 备份恢复 6课时

mysqldump、XtraBackup物理备份、时间点恢复、自动化备份

P6 性能优化 6课时

MONyog监控、压力测试、SQL优化、EXPLAIN分析、阿里云RDS

P7 主从复制 6课时

主从复制配置、半同步复制、并行复制、GTID复制

P8 集群运维 8课时

MySQL Cluster集群部署与管理，双平台集群搭建

P9 Redis运维 8课时

Redis监控、集群创建、MySQL+Redis读写分离实战

P10 智能运维 12课时

goInception DDL审核、自动化运维平台、智能化运维实践
