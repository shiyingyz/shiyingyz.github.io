☷ 功能模块规划（6大模块）

按用户角色和业务领域划分，每个模块包含前端页面 + 后端接口 + 实训步骤文档

M1 用户系统 3页面 / 6接口
  • 注册登录（学号认证 / 手机号 / 微信授权）
  • 个人中心（头像昵称修改、收货地址管理）
  • 我的收藏（商品收藏列表、取消收藏）
  • 信用体系（实名认证、信用评分展示）
  • 安全设置（密码修改、绑定手机、注销账号）
  • JWT Token 无状态认证 + RefreshToken 双 token 机制
M2 商品管理 4页面 / 10接口
  • 商品发布（图片多张上传、分类选择、描述编辑器）
  • 商品详情（大图预览、卖家信息、相似推荐）
  • 商品列表（分类筛选、关键字搜索、价格排序）
  • 我的发布（上架/下架/编辑/删除商品）
  • SPU/SKU 商品模型设计（规格参数+库存）
  • 商品状态机：草稿→审核中→已上架→已售出/已下架
  • Elasticsearch 全文检索（可选增强）
M3 订单交易 4页面 / 12接口
  • 立即购买 / 加入购物车（数量选择）
  • 订单确认页（地址选择、优惠券抵扣、金额计算）
  • 订单详情（物流跟踪、交易状态时间线）
  • 我的订单（全部/待付款/待发货/已完成，Tab 切换）
  • 订单状态机：待付款→待发货→待收货→已完成/已取消
  • 超时自动取消（Redis 过期事件监听）
  • 并发库存扣减（Redis Lua 脚本 / 数据库乐观锁）
M4 支付系统 2页面 / 8接口
  • 支付宝沙箱支付接入（沙箱环境教学用）
  • 微信支付 H5 接入（统一下单 + 回调通知）
  • 支付回调处理（异步通知验签 + 幂等性保障）
  • 退款流程（部分退款 / 全额退款）
  • 交易流水记录（每笔操作可追溯）
  • 支付状态轮询 + WebSocket 推送双通道
M5 消息互动 3页面 / 6接口
  • 即时聊天 IM（WebSocket 双向通信）
  • 消息列表（最新消息预览、未读数角标）
  • 聊天界面（文字/图片发送、历史消息分页加载）
  • 商品评价（买家评价、卖家回复、星级打分）
  • 系统通知（订单状态变更、平台公告推送）
  • 敏感词过滤（正则匹配 + 关键词库）
M6 后台管理 5页面 / 10接口
  • 数据看板（用户/商品/订单/交易额 实时统计图表 ECharts）
  • 用户管理（列表/搜索/禁用/解封/角色分配）
  • 商品审核（违规商品下架、分类管理、举报处理）
  • 订单管理（查看详情、异常订单干预）
  • 内容管理（轮播图配置、公告发布、分类维护）
  • 系统设置（参数配置、操作日志审计）
M7 创业融合元素 4页面 / 课程特色
  • 商业模式画布（九宫格在线填写工具）
  • 盈利分析面板（交易抽成统计、广告位定价模拟）
  • 推广裂变工具（拼团/砍价/邀请码活动设计器）
  • 数据可视化驾驶舱（用户增长漏斗 + GMV 趋势图）
  • 竞品分析矩阵（功能对比表格生成器）
  • 路演数据导出（一键生成 BP 所需核心数据报表）
